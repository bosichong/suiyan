# 本地调试

环境需求：

python (3.6.1)
Django (1.11.3)
Markdown (2.6.8)
Pillow (4.2.1)

进入程序目录后，运行`python3 manage.py runserver`

后台管理地址`http://127.0.0.1:8000/admin/`

管理员登陆:

用户名：`admin`   密码：`17python.com`

前台地址`http://127.0.0.1:8000/`

如果想在部署到生产环境，[请点此查看详细教程](http://www.17python.com/blog/67)
